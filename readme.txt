~~~~~~~~~~~~~~~~Magic text generator~~~~~~~~~~~~~~~~~~~~~~~~

ru

Духи могут писать текст через эту программу.
Просто укажите в качестве аргументов программы 
книги(текст должен быть в кодировке UTF-8), которые 
вы бы хотели использовать в качестве шаблонов для генерирования
 на первый взгляд случайного текста.
НО если вы присмотритесь на текст который выводит программа
в терминал то он окажется со скрытым смыслом.
Можно использовать текст из какого-нибудь чата где очень много текста
тогда текст генерируемый программой будет больше похож на разговор.
